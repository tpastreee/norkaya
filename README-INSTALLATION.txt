NORKAYA - Installation GitHub Pages

1. Dézipper ce dossier.
2. Envoyer tous les fichiers et dossiers à la racine du dépôt GitHub : index.html, css/, js/, images/, etc.
3. Ne pas ouvrir les fichiers directement depuis le ZIP : il faut extraire le dossier complet pour que le CSS et les images se chargent.
4. Sur GitHub Pages, index.html doit être à la racine du dépôt.
